# Simple Study Planner

This is my final project for Python. The project is a simple study planner that helps students calculate an average grade, check how many days are left before a deadline, calculate a task priority score, and create a short study plan sentence.

## Project Files

- `module.py`: contains the main project functions
- `tests.py`: contains unit tests using `unittest`
- `script.py`: runs a short demo of the project
- `requirements.txt`: lists project requirements
- `ProjectNotebook.ipynb`: demonstrates the project, testing, GenAI statement, and extra credit statement

## Functions

The project includes four main functions:

1. `calculate_grade(scores)`  
   Calculates the average grade from a list of scores.

2. `days_left(due_date)`  
   Calculates how many days are left before a due date.

3. `task_priority(importance, days)`  
   Calculates a simple priority score using importance and days left.

4. `make_study_plan(task_name, hours)`  
   Creates a simple study plan sentence.
